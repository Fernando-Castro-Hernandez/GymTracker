# Rediseño visual de GymTracker (rama `06-rediseno-ui`)

Este avance se centró en **renovar por completo el aspecto de la aplicación**.
No se agregó ninguna funcionalidad nueva ni se cambió cómo funciona por dentro:
todo lo que la app ya hacía sigue igual. El objetivo fue que GymTracker **se vea
y se sienta como un producto real de fitness**, no como una plantilla genérica.

## ¿Qué se hizo?

- **Una identidad visual propia.** Se definió una imagen coherente para toda la
  app: un tema oscuro con un color de acento vibrante que transmite la energía y
  la fuerza asociadas al gimnasio. Ese estilo se aplica de forma consistente en
  todas las pantallas.

- **Navegación más clara.** La barra superior se rediseñó para que sea fácil
  moverse entre las secciones (Inicio, Ejercicios y Rutinas), resaltando en qué
  sección estás en cada momento.

- **Página de inicio renovada.** Ahora da la bienvenida explicando de un vistazo
  para qué sirve la app, con accesos directos a lo más importante.

- **Ejercicios y Rutinas más legibles.** Listados, fichas de detalle y
  formularios se reorganizaron para leerse mejor: información ordenada en
  tarjetas, etiquetas de color para identificar cada grupo muscular y botones
  más claros. En el detalle de cada rutina se muestran, además, algunos datos
  rápidos como el total de ejercicios y de series.

- **Mejor experiencia en general.** Mensajes cuando una lista está vacía,
  pantallas que se adaptan al celular y una apariencia más limpia y ordenada.

## Lo importante

- El rediseño fue **solo visual**. La forma de crear, editar y eliminar
  ejercicios y rutinas, el inicio de sesión y el resto del comportamiento se
  conservaron sin cambios.
- El trabajo se hizo **pantalla por pantalla**, de forma ordenada, para poder
  revisar cada cambio por separado.

---

## Uso responsable de la IA

En este avance se utilizó **Claude (Claude Code)** como herramienta de apoyo. Su
uso se limitó a tareas de la capa visual y a la organización del trabajo:

- Apoyo para proponer y aplicar el rediseño de las pantallas y los estilos.
- Ayuda para mantener el cambio ordenado (avanzar por secciones) y para
  verificar que nada del funcionamiento existente se viera afectado.
- Redacción y estructura de este documento.

Las decisiones de diseño, la dirección visual de la aplicación y la revisión
final del resultado son propias. La IA se usó como herramienta de apoyo y
aprendizaje, **no como sustituto del criterio ni del trabajo personal**, y todo
lo entregado fue revisado y comprendido antes de darse por válido.
